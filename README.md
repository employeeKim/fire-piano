# fire-piano
음악학원 웹/앱 플랫폼 프로젝트  
React + Spring Boot 기반의 웹/앱 서비스입니다.

# 사용 기술

## Front-End
- React
- Vite
- JavaScript (ES6+)
- JSX
- Bootstrap

## Back-End
- Spring Boot
- Java 17
- JPA / Hibernate
- MariaDB

# 프로젝트 구조
fire-piano/
 - backend/ # 스프링부트 API 서버
 - frontend/ # 리액트 프론트엔드

# 설치 및 실행 방법
## 1. 백엔드(Spring Boot)
```bash
cd backend
./gradlew bootRun
```

## 2. 프론트엔드(React)
```bash
cd frontend
npm install
npm run dev
```
